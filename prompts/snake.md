# Snake Game - Standard Prompt

## Objective

Create a fully functional **Snake Game** using pure HTML, CSS, and JavaScript in a single file.

## Requirements

### Core Gameplay
- The snake starts at the center of the canvas
- Use arrow keys (↑ ↓ ← →) to control direction
- The snake grows longer when eating food
- Food appears at random positions
- Game over when snake hits walls or itself

### Visual Design
- Canvas size: 400x400 pixels
- Dark background (#0f172a)
- Snake color: cyan (#38bdf8)
- Food color: yellow (#fbbf24)
- Grid-based movement (20x20 cells)

### Game Logic
- Initial snake length: 3 segments
- Movement speed: 100ms per step
- Score counter displayed
- "Game Over" screen with restart option

### Code Structure
- Single HTML file
- Embedded CSS in `<style>` tag
- Embedded JavaScript in `<script>` tag
- No external dependencies

## Deliverable

A complete, working Snake game in one HTML file that can be opened directly in a browser.
